## Run Locally

Clone the project

```bash
  git clone https://github.com/AmirAnckonina/EnglishCheckers.git
```

Go to the project directory

```bash
  cd EnglishCheckers
```

Run Solution ->
```bash
EnglishCheckers.sln
```

Run Executable ->
```bash
/bin/Debug/CheckersUI.exe
```
